# Privacy Policy — dezerata-api

_Last updated: June 30, 2026 · Contact: jmillerfla813@gmail.com_

`dezerata-api` is a personal, single-user software tool operated by Jason Miller to
manage his **own** eBay seller account. This policy explains what data the tool
accesses and how it is handled.

## What the tool does

The tool connects to the operator's own eBay account through eBay's official OAuth
authorization and the eBay Sell APIs (Inventory, Account, and Fulfillment) to create
and publish the operator's own listings and to record shipping/tracking for the
operator's own orders.

## Data we access

The tool accesses only data belonging to the operator's own eBay seller account, using
a token the operator personally authorizes. It does **not** collect, request, or process
the personal data of any other eBay users beyond what eBay returns for the operator's own
orders solely so the operator can ship them.

## Data we store and share

The tool runs locally on the operator's own computer. It does **not** persist, warehouse,
sell, rent, or share any eBay user's personal data with third parties. Buyer and order
details returned by eBay are used transiently to fulfill and ship orders and are not
retained for any other purpose. The operator's own OAuth token is stored only on the
operator's local machine to keep the tool authenticated.

## Marketplace account deletion

Because the tool does not persist eBay users' personal data, there is no stored personal
data to delete in response to an eBay marketplace account-deletion event; the application
is exempt from the account-deletion notification requirement on that basis.
